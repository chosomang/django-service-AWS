How to Search

eventime, actionDisplay, 정규 표현식 등 다양한 검색 방법에 대해 설명합니다.

일반 선택
함수명 설명 예시
eventTime 작업 일시 2022-09-06 14:28:06, 2022-09-04 15:21:18
actionDisplayName 작업(action) 내역 Create ACG, Login
actionResultType 작업(action) 결과 SUCCESS, FAIL
productName 인스턴스 종류(상품명) VPCServer, IAM, Account
resourceType resource 유형 ACG, RouteTable, Group
regioinDisplayName region 한국, KR, 공통
platformType 플랫폼 VPC, BOTH, Classic
actionUserType 계정 구분 Customer, Sub
sourceType 요청 구분 PORTAL, CONSOLE
정규 표현식
기능 설명: actinoDisplayName, EventTime 등 다양한 작업 내역에 대해 정규표현식을 사용해 검색할 수 있습니다.

- 형식1: [작업내역] == /[검색 패턴]/
- 형식2: [작업내역] == [검색어]
[형식 1]은 작업 내역에 대해 검색 패턴이 일치하는 모든 로그를 보여주는 것이며, [형식 2]는 검색어에 정확하게 일치하는 로그를 보여줍니다.

※ 더 다양한 사용법은 기본 정규 표현식을 따라 주세요
함수명 검색어 옵션 예시
&& 또는 and 논리연산자, 모두 동일한 경우 actionDisplayName==abc && resouceType==ACG
== 관계연산자, equal(동일) actionDisplayName==abc
^start start 시작하는 모든 문자열을 매칭 actionDisplayName==/^cr/
end$ end로 끝나는 문자열과 매칭 actionDisplayName==/^cr/
abc abc가 들어가는 모든 문자열 매칭 actionDisplayName==/abc/
abc* ab 그리고 0개 이상의 c 를 포함한 문자열과 매칭 actionDisplayName==/(?i)create/
abc+ ab 그리고 1개 이상의 c 를 포함한 문자열과 매칭 actionDisplayName==/(?i)create/
abc? ab 그리고 0개 또는 1개의 c 를 포함한 문자열과 매칭 actionDisplayName==/(?i)create/
abc{2} ab 그리고 2개의 c 를 포함한 문자열과 매칭 actionDisplayName==/(?i)create/
abc{2,} ab 그리고 2개 이상의 c 를 포함한 문자열과 매칭 actionDisplayName==/(?i)create/
abc{2,5} ab 그리고 2개 이상 5개 이하의 c 를 포함한 문자열과 매칭 actionDisplayName==/(?i)create/
a(bc)* a 그리고 0개 이상의 bc를 포함한 문자열과 매칭 actionDisplayName==/(?i)create/
a(bc){2,5} a 그리고 2개 이상 5개 이하의 bc를 포함한 문자열과 매칭 actionDisplayName==/(?i)create/
a(b|c) 또는 a[bc] a 그리고 b 또는 c를 포함한 문자열과 매칭 actionDisplayName==/(?i)create/
\d 숫자 하나와 매칭 actionDisplayName==/(?i)create/
\w 문자(숫자, 영문, 언더바) 하나와 매칭 actionDisplayName==/(?i)create/
\s 공백문자(탭, 줄바꿈, 스페이스) 하나와 매칭 actionDisplayName==/(?i)create/
. 모든 문자 하나와 매칭 actionDisplayName==/(?i)create/
[abc] 또는 [a-c] a 또는 b 또는 c 를 포함하는 문자열과 매칭 actionDisplayName==/(?i)create/
[a-fA-F0-9] 16진수의 문자 하나와 매칭하고 대소문자를 구분하지 안함 actionDisplayName==/(?i)create/
[0-9]% 0이상 9이하 숫자 그리고 %문자를 포함한 문자열과 매칭 actionDisplayName==/(?i)create/
[^a-zA-Z] 영문이 아닌 문자와 매칭 actionDisplayName==/(?i)create/
(?i) abc 대소문자 구분 없이 매칭 actionDisplayName==/(?i)create/
